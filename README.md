# jquery-tubeground

**Create Youtube-background webpage easily.**

Usage
---
load jQuery and jquery-tubeground.min.js to web project.
create page wrapper (for instance, #wrapper)

    $('#wrapper').tubeground({videoId:'your-video-id'});

and voila! done.
